# ToDo-Firebase

Includes<br/>
expo-sms.   
expo-mail-composer.   
expo-sensors.   

Firebase setup.   
Authentication.   
Firestone.   
Email Verification. 

Getting Started.   
Add your Firebase Config on Firebase.js.   
npm install.   
expo start.    
visit http://localhost:19002/.   
Run on Android or iOS Device.
